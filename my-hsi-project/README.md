# My HSI Project

A Node.js application for tracking and analyzing Hang Seng Index (HSI) data.

## Project Structure

```
my-hsi-project/
├── src/           # Source code
│   └── index.js   # Main entry point
├── config/        # Configuration files
├── data/          # Local data storage
├── package.json   # Project dependencies
└── README.md      # This file
```

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run the Application
```bash
npm start
```
